# API

### auth
`GET /auth/login-check`  
`POST /auth/user-login`  
`POST /auth/user-register`  
`POST /auth/user-logout`  

### folders
`GET /folders`  
`GET /folders/:id`  
`POST /folders`  
`DELETE /folders/:id`  

### tasks
`GET /tasks`  
`GET /tasks/:id`  
`POST /tasks`  
`POST /tasks/:id`  
`DELETE /tasks/:id`  
`GET /tasks/findByFolder/:id`  

### images
`GET /images/:id`
